# Differential Gene Expression Analysis using R

## Project Overview
This project was completed as part of my MPhil Bioinformatics coursework. The aim was to perform differential gene expression analysis using RNA-seq count data from ileal biopsies of pediatric Crohn's disease patients and controls.

## Objectives
- To explore RNA-seq count matrix data.
- To perform preliminary data analysis using PCA and sample-to-sample distance analysis.
- To identify suitable samples for downstream analysis.
- To perform differential gene expression analysis using DESeq2.
- To identify upregulated and downregulated genes.
- To generate a volcano plot for significant differentially expressed genes.
- To perform Gene Set Enrichment Analysis using MSigDB HALLMARK pathways.

## Methods
- RNA-seq count matrix analysis
- Data quality assessment
- PCA analysis
- Sample-to-sample distance heatmap
- DESeq2 differential expression analysis
- Volcano plot generation
- Gene Set Enrichment Analysis
