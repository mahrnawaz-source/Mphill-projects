# Computational Drug Design: Pharmacophore Model Comparison for hERG Inhibitors

## Project Overview
This project was completed as part of the Computational Drug Design course. The project focused on developing and comparing pharmacophore models for identifying potential hERG inhibitors using molecular features, EC50 values, screening hits, and model performance statistics.

## Objectives
- To build pharmacophore models using active compounds.
- To compare different models based on accuracy, specificity, and sensitivity.
- To classify compounds as active or inactive using EC50 threshold values.
- To analyse selected pharmacophore features and distance matrices.
- To evaluate whether the model can distinguish between active and inactive compounds.
- To compare screening hits and conformations across different models.
- To select the most reliable final model for screening unknown data.

## Methods
- Pharmacophore model generation
- EC50-based compound classification
- Feature selection
- Distance matrix analysis
- Screening hit comparison
- RMSD/conformation comparison
- Model performance evaluation
- Final model selection

## Tools and Concepts Used
- MOE
- Pharmacophore modelling
- EC50 activity analysis
- RMSD analysis
- Molecular feature selection
- Virtual screening concepts
- Computational drug design

## Pharmacophore Features Used
- Hydrophobic feature
- Aromatic feature
- Atom Q feature
- Donor and acceptor features
- PiN feature

## Key Findings
- Multiple pharmacophore models were compared.
- Model 1 showed the highest accuracy of 87%.
- Model 1 was selected as the final model.
- The final model included features such as aromatic, Atom Q, and hydrophobic features.
- The model was able to identify active compounds based on low EC50 values.
- Molecule 5 was highlighted as a potential hERG inhibitor due to consistent identification and low EC50 value.
