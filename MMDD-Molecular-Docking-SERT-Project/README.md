# Molecular Modelling and Drug Design: Molecular Docking of SERT Inhibitors

## Project Overview
This project was completed as part of Molecular Modelling and Drug Design coursework. The aim was to perform molecular docking analysis of ligands against the human serotonin transporter (SERT) and compare ligand binding activity, binding affinity, RMSD values, and ligand-protein interactions.

## Objectives
- To define activity threshold criteria for active and inactive ligands.
- To select a suitable protein structure and conformation for docking.
- To prepare receptor and ligand files for docking analysis.
- To define AutoDock Vina search space coordinates.
- To identify key amino acid residues in the binding cavity.
- To compare generated docking poses and binding affinities.
- To interpret RMSD values from AutoDock Vina.
- To relate biological activity with ligand binding pattern.
- To calculate and compare ligand efficiency using pIC50, logP, and LiPE.

## Protein Target
- Target: Human serotonin transporter (SERT)
- Selected PDB ID: 7TXT
- Conformation: Outward-open
- Method: Cryo-electron microscopy
- Resolution: 3.0 Å
