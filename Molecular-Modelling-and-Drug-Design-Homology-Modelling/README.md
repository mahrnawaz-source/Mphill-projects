# Molecular Modelling and Drug Design: Homology Modelling

## Project Overview
This project was completed as part of my Molecular Modelling and Drug Design coursework. The aim was to identify an unknown target protein sequence, find a suitable template protein, build a homology model, and evaluate the quality of the predicted protein structure.

## Objectives
- To identify an unknown protein sequence using BLASTp.
- To retrieve target protein information using UniProt.
- To search and select a suitable template protein from the PDB database.
- To perform target-template sequence alignment.
- To build a homology model using MODELLER.
- To evaluate the predicted protein model using structural validation tools.

## Workflow
1. Target protein identification using BLASTp
2. Protein family and accession information retrieval using UniProt
3. Template search using BLAST/PDB database
4. Sequence alignment using T-Coffee / 3DCoffee
5. Homology model construction using MODELLER
6. Model evaluation using PROCHECK, ERRAT, and Ramachandran plot

## Tools and Databases Used
