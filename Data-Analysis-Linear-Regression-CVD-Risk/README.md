# Data Analysis and Linear Regression Modelling of CVD Risk

## Project Overview
This project was completed as part of my MS/MPhil Bioinformatics coursework for Mathematics for Bioinformatics. The aim was to preprocess health-related data and build simple and multiple linear regression models using CVD Risk as the dependent variable.

## Objectives
- To preprocess the dataset before statistical analysis.
- To handle missing values in the dependent variable.
- To separate blood pressure values into systolic and diastolic blood pressure.
- To detect and remove outliers using Z-scores.
- To examine descriptive statistics and data distribution.
- To build simple linear regression models.
- To build multiple linear regression models.
- To interpret regression coefficients, model fit, and diagnostic plots.

## Methods
- Data preprocessing
- Missing value handling
- Blood pressure column separation into DBP and SBP
- Outlier detection using Z-score rule
- Descriptive statistics
- Boxplot analysis
- Probability plot for normality checking
- Scatterplot analysis
- Simple linear regression
- Multiple linear regression
- Residual diagnostics

## Tools Used
- Minitab
- Microsoft Excel
- Statistical analysis methods
- Regression modelling

## Key Analysis
- CVD Risk was used as the dependent variable.
- Age was selected as a strong predictor for simple linear regression.
- Age and systolic blood pressure were important predictors of CVD Risk.
- Multiple linear regression included variables such as Age, DBP, SBP, BMI, Weight, and Height.
- Model interpretation included R-squared, adjusted R-squared, p-values, coefficients, and residual plots.
