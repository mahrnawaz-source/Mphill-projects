# Next Generation Sequencing Analysis

## Project Overview
This project was completed as part of my MPhil Bioinformatics coursework. It focuses on understanding and applying basic next-generation sequencing analysis preprocessing.

## Objectives
- To understand NGS data analysis steps.
- To perform quality control and preprocessing.
- To practice command-line bioinformatics tools.
- To interpret sequencing analysis results.

## Methods
- FASTQC and FASTP
- Read preprocessing
- Sequence analysis workflow
- Result interpretation

## Tools Used
- Command line
- FACTQC
- fastp
- Bioinformatics workflow concepts

## Skills Demonstrated
- NGS analysis
- Data preprocessing
- Bioinformatics command usage
- Result interpretation
- Scientific report writing
