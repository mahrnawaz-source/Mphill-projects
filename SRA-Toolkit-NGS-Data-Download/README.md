# SRA Toolkit: NGS Data Download

## Project Overview
This project was completed as part of my bioinformatics coursework. The aim was to retrieve sequencing data from the NCBI Sequence Read Archive using the SRA Toolkit and prepare the data for downstream bioinformatics analysis.

## Objectives
- To understand how public sequencing datasets are accessed from NCBI SRA.
- To use SRA Toolkit for downloading raw sequencing reads.
- To convert SRA files into FASTQ format.
- To organise downloaded sequencing data for further quality control and analysis.
- To practise command-line based bioinformatics workflows.

## Methods
- SRA accession search
- Data download using SRA Toolkit
- FASTQ file generation
- File organisation
- Preparation for downstream NGS analysis

## Tools Used
- SRA Toolkit
- NCBI SRA
- Command line / terminal
- FASTQ format
- Bioinformatics workflow concepts

## Skills Demonstrated
- Public biological data retrieval
- NGS data handling
- Command-line bioinformatics
- FASTQ file preparation
- Research data organisation
