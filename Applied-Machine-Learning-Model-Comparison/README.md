# Applied Machine Learning: Performance Comparison of MLP and CNN Models

## Project Overview
This project was completed as part of my MPhil Bioinformatics / Applied Machine Learning coursework. The project focuses on comparing the performance of Multilayer Perceptron (MLP) and Convolutional Neural Network (CNN) models on different datasets.

## Datasets Used
- MNIST
- Iris
- CIFAR-10
- Fashion-MNIST

## Objectives
- To understand the workflow of applied machine learning.
- To build and train MLP models on different datasets.
- To build and train CNN models on image datasets.
- To compare model performance using accuracy, precision, and recall.
- To understand which model is more suitable for image and tabular data.

## Methods
- Data loading and preprocessing
- Normalization of pixel values
- One-hot encoding of class labels
- MLP model development
- CNN model development
- Model training and testing
- Performance evaluation
- Model comparison

## Tools and Libraries Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- scikit-learn

## Models Compared
### Multilayer Perceptron (MLP)
Used for MNIST, Iris, CIFAR-10, and Fashion-MNIST datasets.

### Convolutional Neural Network (CNN)
Used for image datasets including MNIST, CIFAR-10, and Fashion-MNIST.

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Classification report
- Confusion matrix

## Key Findings
- CNN performed better than MLP on image classification datasets.
- MLP was suitable for the Iris dataset because it is tabular data.
- CNN achieved stronger performance on MNIST, CIFAR-10, and Fashion-MNIST.
- The project demonstrated how model choice depends on dataset type.
